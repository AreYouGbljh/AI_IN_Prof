Указатель — это переменная, которая хранит адрес памяти другой переменной.

Основные моменты:
1.  **Объявление:** `int* ptr;` создаёт указатель на тип `int`.
2.  **Операторы:**
    *   `&` (адрес) — возвращает адрес переменной (например, `ptr = &a;`).
    *   `*` (разыменование) — получает значение по адресу (например, `x = *ptr;`).
3.  **Применение:** используются для динамического выделения памяти (`new`/`delete`), работы с массивами, передачи параметров по ссылке и создания сложных структур данных. Позволяют манипулировать памятью напрямую, что даёт гибкость, но требует аккуратности.
